HTML5

Es un lenguaje de etiquetas que sirve para crear la estuctura de la página web. Algunas etiquetas son:

  1. h1...h6
  2. hr
  3. p
  4. ul

CSS

Es un conjunto de reglas que definen el estilo y la apariencia visual de mi página web.

JAVASCRIPT

Es un lenguaje de progrmación que sirve para hacer nuestra página web dinámica.
